\def\Universal{Le Système d'Exploitation Universel}

\def\WhatIsDebianCaption{Qu'est-ce que Debian GNU/Linux?}

\def\WhatIsDebian{
%
Debian est un système d'exploitation (OS) libre pour votre ordinateur. Un
système d'exploitation consiste en l'ensemble des programmes et utilitaires qui
font fonctionner votre ordinateur. Au coeur de chaque système d'exploitation se
trouve le noyau. Il s'agit d'un composant fondamental d'un ordinateur:
il s'occupe de toute la gestion de base et vous permet de démarrer d'autres
programmes. Debian est actuellement basée sur le noyau Linux et inclus plus de
9\,500 packages d'utilitaires et d'applications. Près de 1\,000 développeurs
travaillent dur pour maintenir le haut niveau de qualité de Debian.
%
}

\def\FreedomCaption{Liberté}

\def\Freedom{
%
Debian est entièrement composée de Logiciels Libres. Par Logiciels Libres, nous ne
voulons pas simplement dire gratuits mais également souligner la Liberté
de les utiliser comme bon vous semble, de les partager avec vos amis, de
lire et de modifier leurs codes source et de distribuer ces changements à d'autres
personnes. Cela signifie que Debian peut être utilisée sans limitations, même
dans des environnements commerciaux ! Debian est la plus grande collection de
Logiciels Libres prêt à installer sur Internet.
%
}

\def\CommunityCaption{Communauté}

\def\Community{
%
Le projet Debian est composé à 100\% de volontaires dédiés à la réalisation du
meilleur système d'exploitation possible. Actuellement, près de 1\,000
personnes de par le monde sont occupées à développer le système Debian, avec
des rôles allant du développement de packages à l'assurance qualité (QA), la
sécurité et la stratégie. Le projet Debian s'articule autour de principes de liberté
et d'ouverture des logiciels. Cet engagement est décrit fermement dans le
Contrat Social de Debian publié sur
\boxurl{http://www.debian.org/social_contract}. Les Debian Free Software
Guidelines énoncent les critères que les licences des
logiciels inclus dans le système d'exploitation Debian doivent satisfaire. La
définition de l'Open Source est dérivée de ces lignes de conduite.
%
}

\def\ContinuityCaption{Continuité}

\def\Continuity{
%
Le système de package de Debian permet une migration facile vers de
nouvelles versions des programmes sans exiger de nouvelle installation ni
écraser votre ancienne configuration. Les dépendances entre programmes sont
gérées automatiquement et vous permettent d'effectuer les mises à jour à partir
de tout type de media: disquettes, CD-ROMs, ou directement depuis Internet.
%
}

\def\StabilityCaption{Stabilité}

\def\Stability{
%
Debian ne subit pas de pression commerciale et ne sortira pas de nouvelle et
peut-être instable version simplement parce que le marché l'exigerait. Les
mainteneurs Debian testent continuellement le système et tentent de
supprimer tous les bugs connus avant de sortir une nouvelle version.
%
}

\def\PortabilityCaption{Portabilité}

\def\Portability{
%
Debian est disponible et tourne tout aussi bien sur les architectures
suivantes: Alpha, ARM, HP PA-RISC, IBM S/390, Intel x86, Intel
IA-64, Motorola 68k, MIPS/MIPSel, PowerPC, SPARC.
%
}
