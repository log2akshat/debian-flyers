Debian Flyers

This project maintains several flyers or leaflets for the Debian project.

You can retrieve or browse the full source files used for building multilingual flyers.
Currently, Debian Flyers are available in 9 languages and Debian Social Contract and DFSG in 3 languages.
Translating the general flyer to a new language is as easy as translating the english.tex file and following the building instructions in the README.md file.

We try to offer reasonably up to date copies of typesetted flyers.
