Debian Flyers

This project maintains several flyers or leaflets for the Debian project.

You can retrieve or browse the full source files used for building multilingual flyers. Currently, flyers are available in more than 7 languages. Translating the general flyer to a new language is as easy as translating the english.tex file and following the building instructions in the readme.txt file.

We try to offer reasonably up to date copies of typesetted flyers (PDF and/or PostScript) in our files area.

Mailing List

We use the debian-flyers-devel mailing list for coordination.

Access

This repository is writable by all the Debian Developers. If are not a DD and you need write access, send a mail to the debian-flyers-devel list.
